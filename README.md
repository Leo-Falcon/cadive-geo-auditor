# Cadive GEO Auditor 

A technical tool for analyzing brand visibility in AI-driven search results (ChatGPT, Claude, Perplexity).

## Why this exists
In the "Answer Engine" era, traditional SEO is dying. **Cadive** is building the tools to help brands win the new war for visibility.

## Features
- Semantic Density Analysis
- Entity Disambiguation mapping
- Citation Loop tracking
